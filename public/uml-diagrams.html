Implementation Notesملاحظات التنفيذ

Row Level Security (RLS)امان مستوى الصف (RLS)

All Supabase tables use RLS policies. Users can only read/write their own bookings, trips, saved places, and reviews. Admins (role = 'admin') bypass RLS for management operations. The profiles table allows read access but restricts writes to the authenticated user's own row.تستخدم جميع جداول Supabase سياسات RLS. يمكن للمستخدمين قراءة/كتابة حجوزاتهم ورحلاتهم واماكنهم المحفوظة ومراجعاتهم فقط.

Supabase Auth Integrationتكامل مصادقة Supabase

Authentication uses Supabase Auth with email/password. On sign-up, a database trigger automatically creates a profiles row. Session tokens are stored as HTTP-only cookies managed by the Supabase SSR helper. Middleware checks auth state on protected routes.تستخدم المصادقة Supabase Auth بالبريد الالكتروني/كلمة المرور. عند التسجيل، يقوم مشغل قاعدة البيانات تلقائيا بانشاء صف في الملفات الشخصية.

Next.js App Routerموجه تطبيق Next.js

The app uses the Next.js App Router with Server Components for initial data loading and Client Components for interactive features. Hotel/attraction data is fetched server-side with caching, while filters and trip-building are handled client-side with React state.يستخدم التطبيق موجه تطبيق Next.js مع مكونات الخادم لتحميل البيانات الاولي ومكونات العميل للميزات التفاعلية.

Overpass API Integrationتكامل واجهة Overpass

The "Load More" feature on hotel and attraction pages queries the Overpass (OpenStreetMap) API using bounding box coordinates for the selected city. Results are normalized to match the app's data schema and merged with seed data. Rate limiting and caching prevent excessive API calls.تقوم ميزة "تحميل المزيد" بالاستعلام من واجهة Overpass باستخدام احداثيات المربع المحيط للمدينة المختارة.

Internationalization (i18n)التدويل (i18n)

The app supports English and Arabic. City names are stored with both name (English) and name_ar (Arabic) fields. The UI direction flips to RTL for Arabic. Translation strings are loaded from JSON files organized by locale. The language toggle persists the choice in localStorage.يدعم التطبيق الانجليزية والعربية. يتم تخزين اسماء المدن بحقلي name و name_ar. يتحول اتجاه واجهة المستخدم الى RTL للعربية.
