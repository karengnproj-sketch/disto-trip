Disto-Trip Design System & UX Rationale

Complete documentation of design decisions, component specifications, and user experience rationale for the Egypt Travel Companion web application.

Why this matters: This document explains not just what was built, but why each design decision was made. Every color, component, and interaction pattern was chosen to serve the needs of foreign tourists exploring Egypt - making their journey safer, easier, and more enjoyable.

Visual Identity

Color Palette

11 carefully chosen colors that create a premium, high-contrast dark theme

Why dark theme? (1) Reduces eye strain for travelers browsing at night or in bright sunlight, (2) Makes vibrant Egypt photography pop against the background, (3) Creates a premium feel matching luxury travel brands, (4) Lower battery usage on OLED mobile devices. Why neon green (#39FF14)? High contrast against dark bg (exceeds WCAG AAA 7:1 ratio), connotes 'go/explore/adventure' energy, instantly recognizable brand color.

Inter for everything - readable, multilingual, free

Spacing Scale & Grid

8px base unit for consistent visual rhythm

Why 8px base? Industry standard. Divisible into halves (4px) for fine detail. All spacing tokens are multiples: 4, 8, 12, 16, 24, 32, 48, 64px. Creates mathematical harmony in layouts.

Component Patterns

Reusable components with consistent behavior

Primary Button (CTA)

Style: rounded-full, gradient bg (#39FF14 to #00E676), black text, font-semibold

States: hover: scale(1.05) + shadow-lg + glow | active: scale(0.95) | disabled: opacity-50

Why: Pill shape stands out in dark UI. Gradient creates luminous glow effect. Scale animation mimics physical button press.

Style: bg-#1a1a1a, rounded-2xl (16px), border #333/50, overflow-hidden

Hover: border-color to green/30, shadow-lg, scale(1.02), translateY(-4px)

Why: Large radius feels modern + friendly. Hover lift gives tactile depth. Subtle green glow guides attention to interactive elements.

Style: pl-12 (icon space), rounded-2xl, bg-secondary/80 backdrop-blur

Behavior: Type 2+ chars - dropdown shows cities, hotels, attractions. Enter - full search page.

Why: Instant results reduce friction + bounce rate. Mixed results (cities+hotels+attractions) help users discover. Dropdown closes on blur.

Emergency SOS Button

Style: Large tap target (48px+), red accent override, pulse animation

Behavior: One-tap - tel: link for instant calling. No confirmation needed.

Why: Emergencies require speed. Red overrides green brand color for urgency. Numbers hardcoded (works offline). Large targets prevent mis-taps.

Interactive Map (Leaflet)

Basemap: CartoDB dark tiles (matches app theme, free, no API key)

Markers: Color-coded DivIcons - Blue (hotels), Orange (attractions), Yellow (food), Green (pharmacies), Red (emergency)

Why: Dark basemap is cohesive with theme. Standard OSM would clash. CartoDB is free. Custom markers are instantly recognizable by color.

UX Rationale

Key Design Decisions & Why

Every decision serves the core user: a foreign tourist exploring Egypt

Dark Theme Over Light

Tourists browse while traveling - at night in hotels, in bright sunlight outdoors. Dark theme works in both conditions. It also makes Egypt's golden/warm photography pop dramatically against the dark background.

Emergency First, Always Accessible

Emergency page is always one tap away in the navbar. Numbers are hardcoded (no API dependency, works offline). Red color overrides green brand for urgency. Tourist Police (126) is highlighted - they speak English.

In-Site Booking vs External Redirect

Redirecting to Booking.com breaks the experience. Instead, we built an internal booking form with InstaPay (Egypt's most used payment), cash at hotel, or card at check-in. Keeps users in the ecosystem.

Bilingual EN/AR with RTL

Egypt's tourists are split between English and Arabic speakers. One-click toggle in navbar. RTL layout flips automatically via CSS logical properties and flexbox direction. Arabic font (Cairo) loaded for proper rendering.

Budget-First Planning

Many tourists to Egypt are budget-conscious. The budget planner with 7-currency converter helps them plan costs in their home currency. Typical daily cost tiers (Budget/Mid/Luxury) set expectations before arrival.

Crowd Heuristics Over Real-Time

No free real-time crowd API exists for Egypt. We use time-of-day + day-of-week + season heuristics to estimate crowd levels. Not perfect, but gives tourists useful guidance on when to visit popular sites.

Wikipedia for Landmark Content

Free, comprehensive, multilingual, and always up-to-date. When a user views an attraction, we fetch the Wikipedia summary via REST API. Zero cost, high quality content. Includes thumbnail images.

OpenStreetMap for Live POIs

Google Places API has strict free limits. Overpass API (OpenStreetMap) is completely free and has excellent Egypt coverage for restaurants, pharmacies, banks, mosques. Real-time queries by location.

Framer Motion Animations

Subtle fade-up animations on scroll make the site feel polished without being distracting. Hero section has floating glow orbs. Cards lift on hover. Page transitions are smooth. All animations respect prefers-reduced-motion.

Supabase Over Custom Backend

Supabase provides Auth, PostgreSQL, Row-Level Security, and real-time subscriptions out of the box - all free tier. No need to build and maintain a custom API server. Scales automatically.

Technical Architecture

Tech Stack & Justification

Next.js 15 (App Router)

Server-side rendering for SEO. File-based routing for simplicity. API routes for proxying weather/POI requests. Static generation for fast page loads. Vercel-native deployment.

Tailwind CSS

Utility-first for rapid development. No context-switching between CSS files. Dark mode built-in. Responsive prefixes (md:, lg:). Purged in production for tiny bundle size.

Supabase (PostgreSQL + Auth)

Free tier: 500MB DB, 50K auth users, unlimited API requests. Row-Level Security for data protection. Built-in email auth + Google OAuth. Real-time subscriptions for future features.

Leaflet + OpenStreetMap

100% free, no API key needed. Good Egypt coverage. Dark CartoDB tiles match theme. React-Leaflet for React integration. Custom DivIcon markers for category colors.

Framer Motion

React-native animation library. Scroll-triggered animations (whileInView). Spring physics for natural feel. Layout animations for page transitions. Small bundle impact.

Vercel (Deployment)

Zero-config Next.js deployment. Automatic HTTPS. Global CDN. Preview deployments for PRs. Free tier: 100GB bandwidth, 1M edge requests. Auto-deploys on git push.
